# Before You Start with Git & GitHub 🚀

Before starting your Git & GitHub journey, you need to install Git and set up your environment properly. Git helps you manage files, track changes, and connect your projects with GitHub for version control and collaboration.

## Step 1 — Create a GitHub Account

Visit the official GitHub website and create your account:
https://github.com

## Step 2 — Download Git

Go to the official Git website:
https://git-scm.com

Click on **Download for Windows** and wait for the installer to download.

## Step 3 — Install Git on Windows

1. Open the downloaded Git installer.
2. Click **Next** through the setup options.
3. Keep the default recommended settings.
4. Click **Install**.
5. After installation, click **Finish**.

## Step 4 — Verify Git Installation

Open Command Prompt or VS Code terminal and type:

git --version

If Git is installed correctly, it will display the installed Git version.

## Step 5 — Configure Git

Set your username and email:

git config --global user.name "Your Name"

git config --global user.email "[your_email@example.com](mailto:your_email@example.com)"

## Step 6 — Start Your GitHub Journey

Now you can:

* Create repositories
* Upload files
* Track changes
* Push projects to GitHub
* Manage version control professionally

This is the foundation of Git, GitHub, and modern development workflows. 💻✨
